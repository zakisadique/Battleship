Detailed Description

Represents the player's own grid in the Battleship game.

This class manages the state of the player's own grid, including the placement of ships and processing the impact of shots on the grid.

bool OwnGrid::placeShip ( const Ship &  ship)

Attempts to place a ship on the own grid.

This function checks if the given ship is valid and can be placed on the own grid without overlapping with existing ships. If successful, it updates the state of the own grid by adding the ship and decrementing the ship count.

Parameters
shipThe ship to be placed.
Returns
True if the ship is successfully placed, false otherwise.
